VERIFICATION METHODS
────────────────────

Method 1: Web Interface
   Use this page to upload files and verify.

Method 2: API
   POST /pohw/verify
   {
     "content": "",
     "signature": ""
   }

Method 3: CLI
   $ pohw verify file.txt

Method 4: SDK
   const verification = await pohw.verify(content);

WHAT GETS VERIFIED
──────────────────

✓ Content hash matches
✓ Signature is valid
✓ Signer DID is verified
✓ Attestation exists in registry
✓ Merkle proof is valid
✓ Chain anchors confirm timestamp
✓ No revocation found

PRIVACY
───────

This verification process:
- Does NOT reveal original content
- Does NOT expose identity
- Does NOT track users
- Does NOT store files

Only the hash and signature are used for verification.
